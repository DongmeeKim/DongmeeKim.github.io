### 문자열 연산
-  문자열 더하기 (Concatenation)
	```
	a = "You've got"
	b = "a friend"
	print (a+b)  
		--> You've got a friend 출력 
	```
- 문자열 곱하기 : 문자열의 반복을 의미한다.
	```
	c = "Hello"
	print(c*3)
		--> HelloHelloHello 출력
	```

### 인덱싱과 슬라이싱
-  인덱싱 : 해당 문자열의 값을 가져오는 것
	```
	str = "You've got a friend"
	print(str[4])
		--> v 출력 (0부터 카운트 해야함)
	```

-   슬라이싱 : 문자열 내에 지정하는 부분을 잘라내는 것
	```
	str[-6:] : 인덱스 -6부터 끝까지 슬라이싱하겠다는 의미
	print(str[-6:])
		-->friend 출력
	```
