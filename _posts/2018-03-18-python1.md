## 자료형
-  프로그래밍을 할 때 쓰이는 숫자, 문자열 등등의 것을 뜻한다.

## 자료형의 종류
-  숫자형, 문자열, 리스트, 튜플, 딕셔너리, 집합, 불린(boolean), 변수 등...


### 숫자형
 - 정수 : 12432
 - 실수 : 123.45
 - 복소수 : 1+10i
 - 8진수 : 0o25
 - 16진수 : 0x2F

```
<연산자>
** : 승 값을 나타내는 연산자
	ex)print(a**b)
% : 나머지 값을 반환하는 연산자 
	ex)print(5%2) --> 1 출력
       print(2%5) --> 2 출력
// : 소수점자리를 버리는 연산자
	ex)print(3//2) --> 1출력
```

### 문자열
-  문자들의 집합
-  세 개의 따옴표로 표현하는 경우 (""" 또는 ''')
ex) print("""안녕하세요.
   반갑습니다.
   또 만나요""")

##### 이스케이프
- 여러 줄 짜리 문장을 처리할 때 출력물을 보기 좋게 정렬하기 위해 사용
| 코드 | 출력 결과 |
|----   | ----    |
| \n   | 개행 (줄 바꿈)   |
| \r     | 캐리지 리턴 (커서가 제일 앞으로 돌아가도록 하는 것)   |
| \" | 큰 따옴표 출력 (") |
|\'|작은 따옴표 출력 (')|
|\000 | 널문자       |
|\t | 수평탭       |
|\\ | \ 문자 표현 ('\'와 동일) |
